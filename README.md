# Get Started
### 1. Clone the repository:
```bash
git clone https://github.com/UGoingNoWhereBoy/Vi-Ai
```
### 2. CD into the repository
```bash
  cd Vi-Ai
```
### 3. Install packages 
```bash
npm install
```
### 4. Create a .env file and fill it with your own api key from openai
```bash
touch .env
```
### 5. Run the app 
```bash
npm run dev
```
### 6. Give it a star
